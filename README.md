# Desarrollo de Aplicaciones Web

Este repositorio contiene los ejercicios prácticos de la materia "Desarrollo de Aplicaciones Web".

## 📋 Requisitos Previos

- **Node.js** (v16 o superior)
- **npm** (incluido con Node.js)

## 🚀 Instalación

1. Clona el repositorio:

   ```bash
   git clone <url-del-repositorio>
   cd desarrollo-de-apps-web
   ```

2. Instala las dependencias:
   ```bash
   npm install
   ```

## 🏃 Ejecución

Para ejecutar un ejercicio específico, usa `ts-node`:

```bash
npm start
```

O directamente:

```bash
ts-node index.ts
```

## 📂 Estructura del Proyecto

- `index.ts`: Archivo principal con los ejercicios.
- `package.json`: Configuración del proyecto.
- `tsconfig.json`: Configuración del compilador TypeScript.
